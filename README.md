#Description
- Development environment for Golang project
#How
- share your project with Golang container
```
make run
make jump
```
- delete Golang container
```
make clean
```
#Useful packages
- get godep (dependencies management)
```
apt-get update -y
apt-get install -y git
go get github.com/tools/godep
```
